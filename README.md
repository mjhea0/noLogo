Octoplate
===========

Minimalist boilerplate/theme for [Octopress](http://octopress.org).

Install
-------
    $ cd octopress
    $ git clone git://github.com/mjhea0/noLogo.git .themes/nologo
    $ rake install['nologo']
    $ rake generate && rake preview
    $ rake deploy
